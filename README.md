# Portfólio de Marketing Digital

Este repositório reúne documentações técnicas, playbooks e evidências práticas
de atividades reais executadas em Marketing Digital.

O foco está em **execução técnica**, **organização de processos**, **tracking**,
**automação**, **tráfego pago** e **análise de performance**, sem exposição de
dados sensíveis ou informações de clientes.

## Áreas documentadas
- Tracking e mensuração (GTM, Pixel, GA4, UTMs)
- Tráfego pago (Meta Ads)
- Públicos e remarketing
- Automações (ManyChat)
- Funis, páginas e checkouts
- Diagnóstico técnico e correção de falhas

Cada diretório contém um README explicativo e uma pasta de evidências
com prints anonimizados que comprovam a execução prática.
