# Google Tag Manager (GTM)

Documentação de atividades práticas relacionadas à implementação e manutenção
de tracking utilizando Google Tag Manager.

## Contexto
Páginas e funis sem rastreamento confiável impediam a análise de campanhas,
criação de públicos de remarketing e otimizações baseadas em dados.

## Objetivo
Centralizar a gestão de tags, garantir o disparo correto de eventos e permitir
mensuração consistente de tráfego, leads e conversões.

## Ferramentas utilizadas
- Google Tag Manager
- Google Analytics 4 (GA4)
- Meta Pixel
- DebugView / Tag Assistant

## O que foi feito na prática
- Inserção correta do container GTM no site
- Configuração de eventos padrão e customizados
- Criação de triggers por cliques e interações
- Validação de eventos via DebugView
- Correção de conflitos de tags duplicadas

## Decisões técnicas
- Centralização de tags no GTM para facilitar manutenção
- Uso de eventos padronizados para compatibilidade com plataformas de mídia

## Evidências
A pasta `evidencias/` contém prints anonimizados das configurações e validações.
