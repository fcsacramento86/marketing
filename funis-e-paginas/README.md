# Funis e Páginas

Criação de funis simples para captação e venda.

## Contexto
Processos manuais de venda e conversão.

## Objetivo
Automatizar captação, checkout e entrega.

## Ferramentas utilizadas
- Systeme.io
- Domínio customizado
- Gateways de pagamento

## O que foi feito na prática
- Criação de landing pages
- Configuração de checkout
- Página de obrigado
- Testes de pagamento
