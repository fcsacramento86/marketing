# UTMs e GA4

Criação e uso de parâmetros UTM para rastreamento de origem e desempenho
de campanhas no Google Analytics 4.

## Contexto
Tráfego sem identificação clara de origem, dificultando análise de performance.

## Objetivo
Identificar source, medium e campaign para análise de resultados por canal.

## Ferramentas utilizadas
- UTMs manuais
- Google Analytics 4 (GA4)

## O que foi feito na prática
- Criação de URLs com UTMs para campanhas de tráfego pago
- Análise de sessões por origem e mídia
- Separação de tráfego pago e orgânico

## Decisões técnicas
- Padronização de nomenclaturas para facilitar leitura de relatórios
- Uso do GA4 como fonte principal de análise
