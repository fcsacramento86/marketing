# Automações com ManyChat

Implementação de automações básicas para atendimento e captação de leads.

## Contexto
Atendimento manual e desorganizado via DM do Instagram.

## Objetivo
Automatizar respostas iniciais e organizar leads.

## Ferramentas utilizadas
- ManyChat
- Instagram

## O que foi feito na prática
- Criação de fluxos simples de mensagens
- Configuração de gatilhos por palavra-chave
- Integração com DM do Instagram

## Decisões técnicas
- Automação simples para evitar fricção com o usuário

## Evidências
Fluxos e prints anonimizados em `evidencias/`.
