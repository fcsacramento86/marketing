# Meta Pixel

Implementação e validação de eventos do Meta Pixel para campanhas de conversão
e remarketing.

## Contexto
Campanhas rodando sem dados confiáveis para otimização e reaproveitamento
de tráfego.

## Objetivo
Alimentar corretamente o algoritmo do Meta e permitir criação de públicos
baseados em eventos.

## Ferramentas utilizadas
- Meta Pixel
- Google Tag Manager
- Events Manager

## O que foi feito na prática
- Implementação do Pixel via GTM e/ou código direto
- Configuração de eventos como PageView, ViewContent e Lead
- Diagnóstico e correção de alertas no Events Manager

## Decisões técnicas
- Preferência por implementação via GTM para controle centralizado
- Validação antes da ativação de campanhas de conversão
